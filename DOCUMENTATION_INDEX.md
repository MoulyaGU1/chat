## 📁 Environment & Configuration

### .env.local.example
Template for environment variables. Copy to `.env.local` and fill in:
- `NEXT_PUBLIC_CONVEX_URL`
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`
- `CLERK_SECRET_KEY`

See **QUICKSTART.md** for how to get these values.

## 🤖 Automation Scripts

### install.bat (Windows)
```bash
cd c:\app
install.bat
```
Automated clean installation for Windows.

### install.sh (macOS/Linux)
```bash
cd /path/to/app
bash install.sh
```
Automated clean installation for Unix-like systems.

Both scripts:
- Remove old node_modules
- Clear package-lock.json
- Run fresh npm install
- Show next steps
